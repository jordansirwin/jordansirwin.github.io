Learning Runner Privacy Policy

Information we collect

Learning Runner has no accounts and no collection of information.

Learning Runner has no server infrastructure, it is only a client game running on the device. This website is hosted by GitHub as GitHub Pages with their own independent privacy policies, and we have no responsibility or liability for their content or activities. We do not have access to any of this information if it exists.


Information usage

No information is collected to be used.

Since we do not collect or store any information, there is none to access, change, or delete.


Analytics
